Objects Exercise

For each of the exercises below, assume you are starting with the following programming object:

let programming = {
  languages: ["JavaScript", "Python", "Ruby"],
  isChallenging: true,
  isRewarding: true,
  difficulty: 8,
  jokes: "https://bit.ly/2ysFran"
};

1.  To  access the end of the languages array:  programming.languages[2];
2.  To change difficulty of value to 7:  programming.difficulty = 7;
3.  To delete the jokes key:  delete programming.jokes;
4.  To add a new key call isFun and a value of true to the programming obj:  programming.isFun = true;
